What is this?

This is a package viewer for rew. Here you can search for packages from whatever repos you want.


What is pimmy?

If you have read about rayous, you might have noticed that pimmy is not a command that comes with rew, pimmy is a package manager for rew that can be installed using rew install, it makes it easier to search and manage packages in terminal, read more about it by searching rew.pimmy.
